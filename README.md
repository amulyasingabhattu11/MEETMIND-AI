# 🧠 MeetMind AI

> Transform meeting transcripts into clear summaries, action items & decisions — instantly.

## 🔗 Live Demo
https://meetmind-ai-two.vercel.app

## 🎯 Problem Statement
Professionals waste hours manually summarizing meetings, tracking action items, and following up. MeetMind AI solves this by instantly analyzing meeting transcripts using AI.

## 💡 Solution Overview
MeetMind AI is an intelligent meeting assistant that:
- Generates concise meeting summaries
- Extracts and prioritizes action items with assignees and due dates
- Identifies key decisions made
- Suggests next meeting topics
- Detects meeting sentiment

## 🏗️ Architecture
- **Frontend:** React.js deployed on Vercel
- **AI Engine:** Google Gemini 2.5 Flash API
- **Hosting:** Vercel (CDN)

## 🤖 AI Tools Used
- Google Gemini 2.5 Flash — for transcript analysis and structured output
- GitHub Copilot — used during development

## ⚙️ Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Create `.env` file with `REACT_APP_GEMINI_KEY=your_key`
4. Run `npm start`
